Personal Kanban
You can create new Kanban by selecting Kanban -> New from the Kanban menu. You can give it a name and select number of columns. You can also choose existing Kanban as a template for the new one.
Once Kanban is created you can rename it by clicking it's title in the top bar. You can add new cards to the columns by pressing + in the top right of the column.
You can give each column name and change it's colour by selecting Column settings icon next to it's name. (Number) next to column name indicates number of cards in the column.

Cards are created with title and possible longer description. You can also select different colour for the card if you would like to categorize them somehow. Description will keep the new line formating and add clickable links if you include any.
You can open Card details by clicking on card title.You can edit Card details once opened.
You can drag card between columns and move them up and down column list.

You can have multiple Kanbans and open/switch them from the Switch to ... dropdown, next to Cloud Menu.
You can select different style of Kanban board from Kanban -> Theme menu.
You can delete entire Kanban by selecting Kanban -> Delete from the menu.
Your Kanbans are automatically saved in your browser storage.

If card is in the last colum you can Archive it. It will remove it from a Column but keep in Kanban. You can review your archived Cards via Kanban -> Archives menu.
You can also un-archive cards from Archive and permamently remove them.

You can Export one or all Kanbans by selecting Kanban -> Export menu.
You can Import Kanban(s) from a JSON file. Not that Kanban with existing name, same as imported one will be replaced with the content of the new one.

Cloud
You can use My Personal Kanban Cloud service that enables Upload and Download from Cloud. This service is offered free of charge.
You will need to create your cloud Kanban key. Follow instructions on Cloud -> Setup.
Your Kanban in the Cloud is stored Encrypted, that's why you should setup Encryption Key. The Encryption algorithm is Rabbit.
You can upload Kanban by selecting Cloud -> Upload from the menu
. You can download Kanban by selecting Cloud -> Download from the menu.

You can also use your own Local Cloud. There is server implementation provided at: https://github.com/greggigon/my-personal-kanban-server. Just select the option and provide URL for the Local Cloud service Cloud -> Setup -> Use own Local Cloud.
